MEMÓRIA RAM

O que é memória RAM e qual função? A memória RAM é uma tecnologia que permite o acesso aos arquivos armazenados no celular, PC ou notebook. Diferentemente da memória do HD, a RAM não armazena conteúdos permanentemente. O componente é responsável pela leitura dos conteúdos quando requeridos. é possível fazer uma analogia com uma mesa de estudos. Nela, há todo o material necessário para realizar os deveres de casa, como canetas, lápis, caderno e livros. Os materiais seriam os arquivos, enquanto a memória RAM seria a mesa, onde tudo se reúne e o trabalho é feito.
